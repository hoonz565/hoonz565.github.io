# Nguyễn Minh Hưng - Software Engineer Portfolio 🚀

Welcome to the repository of my personal portfolio. This project is a modern, high-performance web application designed to showcase my skills, experience, and projects as a Software Engineering student at Ho Chi Minh City University of Technology (HCMUT).

**Live Demo:** [https://mhung.github.io](https://mhung.github.io)

---

## ✨ Design Concept

Inspired by the clean, sophisticated aesthetic of modern developer portfolios (notably Brittany Chiang's v4 design), this portfolio focuses on a sleek dark theme, high-quality typography, and smooth interactive elements. It provides a seamless user experience that highlights technical proficiency through both content and implementation quality.

### Key Features
* **Modern Dark UI:** A professional color palette (Navy/Slate/Green) optimized for readability and aesthetics.
* **Smooth Animations:** Powered by **Framer Motion** for polished transition and scroll-reveal effects.
* **Responsive Design:** Fully optimized for all screen sizes, from mobile devices to ultra-wide monitors.
* **Interactive Components:** Custom-built navigation, project highlights, and social sidebar integrations.
* **Performance Optimized:** Built with React 19 and Vite for lightning-fast load times and smooth interactions.

---

## 🛠️ Tech Stack

This project leverages cutting-edge web technologies to ensure performance and maintainability:

* **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) (using the latest features for optimized builds)
* **Animation:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Deployment:** [GitHub Actions](https://github.com/features/actions) & [GitHub Pages](https://pages.github.com/)

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hoonz565/mhung.github.io.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd mhung.github.io
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
